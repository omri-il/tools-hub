QR דינמי

קוד QR אחד שאפשר לשנות לאן הוא מפנה בכל רגע — בלי להדפיס מחדש. היעד נשמר בתא בגוגל שיטס שלכם; משנים את התא, וה-QR כבר מפנה למקום החדש. הכל חינמי, בלי שרת שלי, רץ אצלכם.

1

צרו את הקוד


        


        
    
          
  1. פתחו Google Sheet חדש (אפשר להקליד sheets.new בדפדפן).
    2. 
          
  2. בתפריט: Extensions → Apps Script (הרחבות → Apps Script).
    3. 
          
  3. מחקו את הקוד שכבר נמצא שם, הדביקו את הקוד שלמעלה, ושמרו (💾).
    4. 
          
  4. למעלה בחרו את הפונקציה setup ולחצו ▷ Run. אשרו הרשאות בפעם הראשונה. זה יוצר גיליון בשם QR עם כתובת היעד בתא B1.
    5. 
          
  5. לחצו Deploy → New deployment → סוג Web app → Execute as: Me → Who has access: Anyone → Deploy. העתיקו את כתובת ה-Web app (מסתיימת ב-/exec).
    6. 
          
  6. הדביקו אותה בשלב 2 למטה וצרו את ה-QR.
    7. 
        

        
💡 איך משנים יעד בעתיד: פותחים את הגיליון, עוברים לגיליון QR, ומשנים את תא B1. זהו — ה-QR המודפס כבר מפנה למקום החדש.
