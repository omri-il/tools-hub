מחולל אימייל אוטומטי לטופס

כותבים את המכתב פעם אחת, מקבלים קוד מוכן — וכל מי שממלא את טופס ה-Google שלכם מקבל אותו מיד, אוטומטית. בלי לכתוב שורת קוד.

א.

תוכן המכתב

העלו את התמונה לאתר או ל-Google Drive (קישור ישיר) והדביקו כאן. תופיע במרכז בראש המייל.

עמודה A = חותמת זמן, עמודה B = אימייל. לכן השאלה הראשונה בטופס היא בדרך כלל עמודה C.

🔒 אם תמלא — הקוד יוסיף משתתפים אוטומטית למעקב 5 הימים. הכתובת נשמרת רק במחשב שלך ולא מתפרסמת בדף.

🔗 כל כפתור במייל יעבור דרך מעקב, ייכתב לגיליון בלשונית Clicks (נמען · קישור · זמן), וימשיך מיד ליעד. צריך לפרסם את הסקריפט כ-Web App — ההוראות יופיעו מתחת לקוד.

ב.

כך ייראה המייל

ג.

הקוד המוכן להדבקה



      

        איך מתקינים את הקוד? חמישה שלבים
        
    
          
  1. בטופס Google שלכם — ודאו שמופעל "Collect email addresses" (האימייל נקרא מעמודה B).
    2. 
          
  2. פתחו את הגיליון המחובר לטופס → תפריט Extensions → Apps Script.
    3. 
          
  3. מחקו את הקוד הקיים, הדביקו את הקוד מכאן, ולחצו על 💾 לשמירה.
    4. 
          
  4. הריצו פעם אחת את הפונקציה sendTestEmail ואשרו את ההרשאות — יישלח אליכם מייל בדיקה.
    5. 
          
  5. הגדירו טריגר: סמל השעון ⏰ (Triggers) → Add Trigger → פונקציה sendEmailOnFormSubmit, מקור From form, סוג On form submit → שמרו.
    6. 
        

        
זהו. מעכשיו כל מי שימלא את הטופס יקבל את המכתב אוטומטית.

      


      

        📊 מעקב קליקים מופעל. כדי שזה יעבוד, פרסמו את הסקריפט כ-Web App:
        Deploy ← New deployment ← סוג Web app ← Execute as: Me ← Who has access: Anyone ← Deploy, ואשרו הרשאות.
        התוצאות ייכתבו אוטומטית בגיליון שלכם בלשונית Clicks: זמן · קמפיין · נמען · קישור.
