ממלאים את הפרטים, רואים תצוגה חיה של המייל, ומקבלים קוד מוכן להדבקה ב-Google Apps Script. כל מי שימלא את הטופס יקבל את המייל אוטומטית — בלי לכתוב שורת קוד.

פרטי המייל

כל שינוי כאן מתעדכן מיד בתצוגה ובקוד שמשמאל.

עמודה A = חותמת זמן, עמודה B = אימייל. לכן השאלה הראשונה בטופס היא בדרך כלל עמודה C.

🔒 אם תמלא — הקוד יוסיף משתתפים אוטומטית למעקב 5 הימים ב-MailerLite. הכתובת נשמרת רק במחשב שלך ולא מתפרסמת בדף.

הקוד המוכן להדבקה



      

        איך מתקינים את הקוד? (5 שלבים פשוטים)
        
    
          
  1. בטופס Google שלכם — ודאו שמופעל "Collect email addresses" (האימייל נקרא מעמודה B).
    2. 
          
  2. פתחו את הגיליון המחובר לטופס → תפריט Extensions → Apps Script.
    3. 
          
  3. מחקו את הקוד הקיים, הדביקו את הקוד מכאן, ולחצו על 💾 לשמירה.
    4. 
          
  4. הריצו פעם אחת את הפונקציה sendTestEmail ואשרו את ההרשאות — יישלח אליכם מייל בדיקה.
    5. 
          
  5. הגדירו טריגר: לחצו על סמל השעון ⏰ (Triggers) → Add Trigger → בחרו פונקציה sendEmailOnFormSubmit, מקור From form, סוג On form submit → שמרו.
    6. 
        

        
זהו! מעכשיו כל מי שימלא את הטופס יקבל את המייל אוטומטית. 🎉
