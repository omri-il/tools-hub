מחולל אימייל אוטומטי לטופס

כותבים את המכתב פעם אחת, מקבלים קוד מוכן — וכל מי שממלא את טופס ה-Google שלכם מקבל אותו מיד, אוטומטית. בלי לכתוב שורת קוד.

א.

תוכן המכתב

🎨 עיצוב ושולח — צבע, גופן, לוגו, חתימה
או הדביקו קישור למעלה

⚠️ הקובץ גדול יחסית — הקוד שנוצר יהיה ארוך. כדאי להעלות תמונה קטנה או פשוטה יותר.

אפשר להדביק קישור ישיר לתמונה (אתר / Google Drive) או להעלות קובץ מהמחשב — קובץ שתעלו מוטמע ישירות בקוד כדי שיוצג בכל תוכנת דואר. יופיע במרכז בראש המייל.

הדביקו קישור לקובץ ב-Drive (PDF / מצגת / תמונה). שתפו אותו כ"כל מי שיש לו הקישור — צופה". הקובץ יצורף לכל מייל. הקוד ישתמש ב-DriveApp ולכן יבקש הרשאת Drive בפעם הראשונה.

⚙️ אפשרויות מתקדמות פעיל — עמודת אימייל, פנייה בשם, מעקב

עמודה A היא תמיד חותמת הזמן. אם הפעלתם "Collect email addresses" בטופס — האימייל בעמודה B, השאירו כברירת מחדל. אם ביקשתם את האימייל כשאלה רגילה בטופס — פתחו את הגיליון, בדקו באיזו עמודה מופיעה כתובת המייל, ובחרו אותה כאן.

פתחו את הגיליון ובדקו באיזו עמודה מופיע השם, ובחרו אותה כאן. (עמודה A = חותמת זמן; אם האימייל נאסף אוטומטית הוא בעמודה B, והשאלה הראשונה בטופס היא עמודה C.)

🔒 אם תמלא — הקוד יוסיף משתתפים אוטומטית למעקב 5 הימים. הכתובת נשמרת רק במחשב שלך ולא מתפרסמת בדף.

🔗 כל כפתור במייל יעבור דרך מעקב, ייכתב לגיליון בלשונית Clicks (נמען · קישור · זמן), וימשיך מיד ליעד. צריך לפרסם את הסקריפט כ-Web App — ההוראות יופיעו מתחת לקוד.

📨 אימייל מעקב אוטומטי פעיל — נשלח כמה ימים אחרי מילוי הטופס
📨 מעקב ראשון
📨 מעקב שני

ג.

הקוד המוכן להדבקה



      

        איך מתקינים את הקוד? חמישה שלבים
        
    
          
  1. ודאו שהטופס אוסף כתובות אימייל — או דרך "Collect email addresses" (אז האימייל בעמודה B), או כשאלה רגילה בטופס. אם האימייל אינו בעמודה B, בחרו את העמודה הנכונה תחת ⚙️ אפשרויות מתקדמות → עמודת האימייל.
    2. 
          
  2. פתחו את הגיליון המחובר לטופס → תפריט Extensions → Apps Script.
    3. 
          
  3. מחקו את הקוד הקיים, הדביקו את הקוד מכאן, ולחצו על 💾 לשמירה.
    4. 
          
  4. הריצו פעם אחת את הפונקציה sendTestEmail ואשרו את ההרשאות — יישלח אליכם מייל בדיקה.
    5. 
          
  5. הגדירו טריגר: סמל השעון ⏰ (Triggers) → Add Trigger → פונקציה sendEmailOnFormSubmit, מקור From form, סוג On form submit → שמרו.
    6. 
        

        
זהו. מעכשיו כל מי שימלא את הטופס יקבל את המכתב אוטומטית.

      


      

        📊 מעקב קליקים מופעל. כדי שזה יעבוד, פרסמו את הסקריפט כ-Web App:
        Deploy ← New deployment ← סוג Web app ← Execute as: Me ← Who has access: Anyone ← Deploy, ואשרו הרשאות.
        התוצאות ייכתבו אוטומטית בגיליון שלכם בלשונית Clicks: זמן · קמפיין · נמען · קישור.
      


      

        📨 אימייל מעקב מופעל. כדי שזה יעבוד, צריך עוד שני צעדים חד־פעמיים:

        1️⃣ הריצו פעם אחת את הפונקציה sendFollowUps (מאשר הרשאות + יוצר בסיס — מי שכבר מילא לא יקבל מעקב למפרע).

        2️⃣ הגדירו טריגר יומי: סמל השעון ⏰ ← Add Trigger ← פונקציה sendFollowUps ← מקור Time-driven ← Day timer ← שמרו.

        💡 לבדיקת התוכן לפני הכל, הריצו sendFollowUpTest — יישלח אליכם עותק של כל מייל מעקב.
